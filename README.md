# Express + Swagger Example

This project is a minimal Express app with Swagger (OpenAPI) documentation.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open Swagger UI:

http://localhost:3000/api-docs

Files created

- `index.js` - Express server and Swagger setup
- `routes/users.js` - Sample route with JSDoc/OpenAPI comments

Notes

- To run with auto-reload during development, install `nodemon` globally or use `npm run dev`.
